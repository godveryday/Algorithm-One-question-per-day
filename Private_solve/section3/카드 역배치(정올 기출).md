# 카드 역배치(정올 기출)  

#### 내 풀이

```
arr = [i for i in range(21)]
    
for i in range(10):
    start, end = map(int, input().split())
    while(start != end and start<end):
        arr[start], arr[end] = arr[end], arr[start]
        start+=1
        end-=1

arr.remove(0)
print(arr)
 
```

#### 강사 풀이

```
a = list(range(21))

for _ in range(10):
  s, e = map(int, input().split())
  for i in range((e-s+1)//2):
    a[s+i], a[e-i] = a[e-i], a[s+i]

a.pop(0)
for x in a:
  print(x, end=' ')
```
