# 소수(에라토스테네스 체) 15min 

--> 이중 for문 사용해서 틀림


#### 내 풀이

```
cnt = 0
N = int(input())
for i in range(2, N+1):
    if any(i % x ==0 for x in range(2, i)):
        continue
    cnt+=1
    
print(cnt)  
 
```





#### 강사 풀이

```
n = int(input())
ch = [0] * (n+1)      # index번호가 입력되는 거 까지 생겨야함
cnt = 0

for i in range(2, n+1):
    if ch[i] ==0:
        cnt+=1
        for j in range(i, n+1, i):
            ch[j]=1

print(cnt)

```
