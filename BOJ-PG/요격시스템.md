#### 개구간 (a,b) == a<x<b

```
def solution(targets):
    answer = 0
    targets.sort(key=lambda x:x[1])
    
    pivot = 0
    for s, e in targets:
        if s>=pivot:
            answer+=1
            pivot = e
    return answer
```
